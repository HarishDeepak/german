# Skill: write
**Use this skill for:** writing practice, text correction, B1 writing tasks, learning to write emails, letters, opinions, and short essays in German.

---

## What this skill does
Guides me through structured German writing tasks. I write, Claude corrects systematically, explains errors, and shows me improved versions. Over time I learn to write accurately and naturally.

## How to activate
- `/write` — Claude picks a task based on B1 targets or weak spots
- `/write email` — formal or informal email/letter
- `/write Meinung` — opinion text on a topic (B1 exam style)
- `/write Bewerbung` — job application text
- `/write Beschreibung` — describe a picture, a place, or a person
- `/write korrigier` — I paste a text I wrote and ask for correction only
- `/write [free topic]` — I choose the topic

---

## Session structure

### Step 1: Task setup
Claude gives a clear writing prompt with:
- Word count target (B1 tasks: typically 80–150 words)
- Register: formal (Sie) or informal (du)
- Required elements to include (to make it realistic)

### Step 2: I write
I write the full text in German. No help yet.

### Step 3: Correction
Claude goes through my text and marks:

**Error types:**
- 🔴 Grammar error (article, case, verb form, word order)
- 🟡 Unnatural phrasing (correct but a German wouldn't say it)
- 🟢 Good — natural German, keep this

Claude shows a corrected version with inline notes.

### Step 4: Improved model text
Claude writes a model answer at ~my level + 20% (i+1 principle).
I can compare mine to the model.

### Step 5: Focus point
One key lesson from this writing session: the most repeated error or the most impactful thing to fix.

## B1 writing task types

### 1. Informal email / message (häufigster Prüfungstyp)
- To a friend: suggest plans, react to news, describe an experience
- Must include: greeting, reason for writing, 2–3 content points, closing

### 2. Formal email / letter (formal register)
- To a company, landlord, office
- Must include: formal greeting (Sehr geehrte/r...), clear purpose, polite request, formal closing (Mit freundlichen Grüßen)

### 3. Opinion / comment (Stellungnahme)
- Topic + prompt: "Was meinen Sie dazu?"
- Must include: clear position, 2 reasons or examples, counter-argument, conclusion
- Key connectors to use: Meiner Meinung nach, Einerseits…andererseits, Obwohl, Deshalb, Insgesamt

### 4. Short description / story (Beschreibung)
- Describe a picture, a place, a recent experience
- Must include: structured description, adjectives, past tense (Perfekt)

## Useful writing phrases

**Starting a formal email:**
- Sehr geehrte Damen und Herren,
- Sehr geehrte Frau [Name],
- Sehr geehrter Herr [Name],
- Ich schreibe Ihnen, weil…
- Ich wende mich an Sie bezüglich…

**Starting an informal email:**
- Liebe/r [Name],
- Hey [Name],
- Ich hoffe, es geht dir gut.
- Ich wollte dir kurz schreiben, weil…

**Structuring an opinion:**
- Meiner Meinung nach / Ich finde, dass…
- Ein wichtiger Grund dafür ist, dass…
- Außerdem / Darüber hinaus…
- Zwar…, aber… / Einerseits…, andererseits…
- Obwohl ich verstehe, dass…, bin ich trotzdem der Meinung, dass…
- Insgesamt / Zusammenfassend lässt sich sagen, dass…

**Closing formal:**
- Ich würde mich über eine baldige Antwort freuen.
- Für Rückfragen stehe ich gerne zur Verfügung.
- Mit freundlichen Grüßen,

**Closing informal:**
- Ich freue mich auf deine Antwort.
- Bis bald! / Liebe Grüße,

## Common writing errors (A2 learners)
1. Wrong verb position in subordinate clauses (*Ich denke, dass er **kommt** nicht* → er **nicht kommt**)
2. Wrong case after prepositions
3. Missing adjective endings
4. Using English-style word order
5. Using present tense instead of Perfekt for past events
6. Mixing formal and informal register
7. Too-short sentences (no connectors)
